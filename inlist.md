# inlist

Inlist is a simple format for writing structured text.

```
agent
  name
    Mitch Henderson
  role
    hotdog seller
  inventory
    hotdog cart
    tongs
    hat
      attribute
        striped
  description
    this hotdog seller wants to sell you a hotdog

agent
  name
  	Johnson
  role
  	newspaper reader
  inventory
  	newspaper
  description
  	Johnson is sitting on a bench, reading a newspaper
```

It is supported by the Termpose suite
